# synent-task1-Personal-Portfolio-Website-amirtha
 Personal Portfolio Website

## Project Objective

The objective of this project is to create a personal portfolio website that showcases my profile, skills, and contact information in a clean and professional manner.

## Steps Performed

1. Designed a single-page portfolio website layout.
2. Created a Header section with my name and title.
3. Added an About Me section to introduce myself.
4. Included a Skills section to highlight my technical skills.
5. Added a Contact section for communication details.
6. Applied CSS styling to ensure proper spacing, alignment, and visual appearance.
7. Tested the website for responsiveness and usability.

## Tools Used

* HTML
* CSS
* Visual Studio Code
* GitHub

## Outcome

Successfully developed a clean and professional single-page portfolio website with Header, About Me, Skills, and Contact sections, ensuring proper layout, spacing, and alignment.
